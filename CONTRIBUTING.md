# Contributing to LinkedIn Automation Tools Directory

Thank you for your interest in contributing! This repository is automatically synced with the main [LinkedAutomation](https://linkedautomation.org) directory, but we welcome contributions for corrections and improvements.

## How to Contribute

### Reporting Issues

If you find an error in any tool listing:

1. **Check existing issues** first to avoid duplicates
2. **Open a new issue** with:
   - Tool name
   - Description of the error
   - Correct information (with sources if possible)
   - Screenshot (if applicable)

### Suggesting New Tools

To suggest a new tool for inclusion:

1. **Open an issue** with the title format: `Add: [Tool Name]`
2. Include the following information:
   - Tool name
   - Website URL
   - Category (automation-workflows, content-generation, etc.)
   - Pricing model
   - Brief description
   - Why it should be included

### Pull Requests

For direct contributions:

1. **Fork the repository**
2. **Create a feature branch**: `git checkout -b update/tool-name`
3. **Make your changes** following our formatting guidelines
4. **Submit a pull request** with a clear description

### Writing Style

- **Be objective**: Present facts, not opinions
- **Be concise**: Keep descriptions under 100 words
- **Be accurate**: Verify all pricing and features
- **Be helpful**: Include use cases and best practices

## Code of Conduct

### Our Standards

- Use welcoming and inclusive language
- Be respectful of differing viewpoints
- Accept constructive criticism gracefully
- Focus on what's best for the community

### Unacceptable Behavior

- Trolling, insulting/derogatory comments
- Public or private harassment
- Publishing others' private information
- Other conduct which could reasonably be considered inappropriate

## Review Process

All contributions go through a review process:

1. **Automated checks** - Verify formatting and links
2. **Manual review** - Check accuracy and relevance
3. **Approval** - Merge or request changes

Typical response time: 3-5 business days.
